
# Quick Reference – ScoreMyResume

Run:
streamlit run streamlit_app.py

Workflow:
1. Upload resume
2. Add JD
3. Run analysis
4. Review results

Scores:
1. 75–100 Strong
2. 60–74 Good
3. 40–59 Fair
4. 0–39 Poor

Junior: projects count fully
Mid: projects 50%
Senior: only professional experience

Python evidence:
skills, projects, experience, frameworks, metrics
